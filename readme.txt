* Install Python & Packages *

1) Install Anaconda (Python package manager)
> https://www.continuum.io/downloads 
> Anaconda for Windows 
> PYTHON 2.7 
> Click "Windows Graphical Installer" and install

2) (Only for GUI usage) Install GUI packages
> Run cmd
> type "conda install PyQt"


* Functionalities *
This program has 4 functionalities

1) Run in GUI mode
do :
    python gui.py

2) Convert s2p to mdif, or Convert mdif to s2p

3) De-embed s2p

4) Help
do :
    python run.py -h


Test Run :
python parser.py
python de_embeder.py
python converter.py
python run.py
